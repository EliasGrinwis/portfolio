DevOps (ASP.NET WEB APP)

Achtergrond informatie

In dit project heb ik gekozen om met contentful te werken. Ik heb een ASP.NET applicatie geschreven dat aan de hand van een powershell script enkele gegevens over mijn volgers binnen haalt. Zoals een naam, gebruikersnaam en hun id. Ook gebruik ik hun profielfoto. Jammer genoeg heb ik met mijn powershell script niet de profiel foto’s kunnen krijgen en heb ik deze handmatig in mijn applicatie gezet. Deze gegevens stuur ik dan allemaal door naar mijn contentful. Ten slotte laat ik deze gegevens zien in een browser. Mijn applicatie wordt ook gehost op Okteto. In het filmpje laat ik de hele werking zien.

Youtube video

Besluit

Dit project omvatte het gebruik van Contentful om gegevens van volgers te beheren en te tonen in een ASP.NET applicatie. Ondanks enkele uitdagingen, slaagde de applicatie erin om de gegevens te verzamelen en te presenteren in een browser. Het project was zeer leerzaam en bood de kans om nieuwe vaardigheden en inzichten op te doen.
